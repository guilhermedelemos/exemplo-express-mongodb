# Exemplo Express+MongoDB

## Executar
* `npm install`
* `npm run dev`

## MongoDB

### Docker
* `# docker pull mongo`
* `# docker run --name mongodb -p 27017:27017 -d mongo`
* `# docker ps`

--name nome do container dentro do sistema

-p mapeamento de portas (local:container)

-d nome da imagem a ser usada
